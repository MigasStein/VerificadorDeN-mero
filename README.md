# Verificador de Número em Java

Você digita um número e ele te conta tudo sobre ele — se é par, primo, perfeito, quadrado perfeito... Feito para praticar lógica e funções em Java.

**Exemplo**

Digite um número inteiro: 28

--- Resultados ---
Par ou ímpar : Par

Positivo/Neg : Positivo

Primo        : Não

Perfeito     : Sim  ✅ (1+2+4+7+14 = 28)

Quadrado     : Não
